# Med-Tech-HMIS
HMIS Web Application
This is a Health Management Information System built to run in the cloud.
The project will adapt a modular way of implementation; where each feature is implemented as a component.

## Modules
- Patients
- Visits
- Self Requests
- Tringe
- Doctor
- Lab
- Radiology
- Dental
- Pharmacy
- Theater 
- Inventory
- Finances 
- Cashier
- Invoices
- SMS

## Languages Used

| FrontEnd   | BackEnd    |
| ---------- | ---------- |
| HTML       |   Django   |
| CSS        |   Nodejs   |
| JAVASCRIPT |  Postgres  |
